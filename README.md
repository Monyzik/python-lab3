# Лабораторная работа №3

В лабораторной работе реализован простой CLI интерфейс,
для его запуска необходимо запустить команду:

```shell
python -m src.main
```

### Команды которые доступны для пользователя

##### Доступные команды можно получить прописав:

```shell
help
```

##### Вычисление факториала:

```shell
Usage: factorial [OPTIONS] N

Options:
  -r
  --help
```

##### Вычисление n-ого числа Фибоначчи:

```shell
Usage: fibonacci [OPTIONS] N

Options:
  -r
  --help
```

##### Использование сортировок:

```shell
Usage: sort [OPTIONS] {bubble|bucket|counting|heap|quick|radix} [ARRAY]...

Options:
  --base INTEGER
  --buckets INTEGER
  --help
```

##### Работа со стеком:

```shell
Usage: stack [OPTIONS] COMMAND

Options:
  -i, --item INTEGER
  --help
```

Доступные команды: push, pop, size/len, peek, is_empty, min

##### Работа с очередью:

```shell
Usage: queue [OPTIONS] COMMAND

Options:
  -i, --item INTEGER
  --help
```

Доступные команды: enqueue, dequeue, front, size/len, is_empty

##### Выход из CLI:

``` shell
exit
quit
```
